tbfac.newcontent Installation
-----------------------------

To install tbfac.newcontent using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``tbfac.newcontent`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        tbfac.newcontent
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
